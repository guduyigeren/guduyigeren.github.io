Java 语言入门

一、变量

  • 1.数据类型

    基本数据类型定义了4类共8种基本数据类型分别为：

    布尔型：boolean。

    字符型：char。

    整数型：byte、short、int、long。

    浮点型：float、double。

  • 2.数据进制

    布尔型：boolean：

    • boolean类型通常用于逻辑运算和程序流程控制。

    • boolean类型的数据只允许取值true或false。

    字符型char：

    • char型数据用来表示通常意义上的"字符"

    • char占2字节，范围是[0,65535]，但是前256个表示特殊字符。

    • 字符常量为用单引号括起来的单个字符，比如：

      char a = 'B'

    • java字符采用Unicode编码，可用十六进制编码形式表示。

    整数类型byte、short、int、long：

    • 二进制整数，0b或0B开头，如：int a = 0B110；

    • 十进制整数，如：int a = 17;

    • 八进制整数，要求以0开头，如 int a = 012;

    • 十六进制整数，要求0X或0x开头，如int a = 0x12;

    浮点类型float、double：

    • double和float并不能表示一个精确的数字

    • Java浮点类型常量有两种表示方式：

      • 十进制形式： 例如：3.14

      • 科学计数法形式： 例如：3.14e2

    • Java浮点型常量默认是double，则要声明一个常量为float型，则需在常量后加上f 或 F,如：

      • float a = 3.14F;

  • 3.标识符

    • 1.可以简单理解为在java程序中为了增强阅读性自定义的名称。如：类名、方法名、变量名等。

    • 2.命名规则：

      • 1.由字母、数字、下划线、$组成，不能以数字开头

      • 2.大小写敏感

      • 3.不得使用java中的关键字和保留字

      • 4.别用Java API里面的类名作为自己的类名

  • 4.类型转换

    • 1.boolean 类型不能转换成为其他基本数据类型。

    • 2.整形、字符型、浮点型的数据可以在混合运算中相互转换。

      • 1.容量小的类型可以自动转换为容量大的数据类型

      • 2.byte、short、char → int → long → float → double

      • 3.byte、short、char之间不能互相转换

    • 3.容量大的数据类型转换为容量小的数据类型时，要加上强制转换符。

    • 4.有很多种类型的数据混合运算时，系统首先自动的将所有数据转换成容器最大的哪一种数据类型，再计算

    • 5.浮点型：默认是double，整型：默认是 int 。

二、Java基本语法格式

  • 1.常量

    • 常量：程序中固定不变化的值。比如：整数1,2,3，小数3.14,false,true等。
  • 2.关键字

关键字(都是小写的)是电脑语言里事先定义的,有特别用途的单词

  • 3.转义字符

\t 制表符
\r 回车符
\" 双引号
\ 反斜杠
\' 单引号

三、运算符

  • 算术运算符
运算符 运算 范例 结果
+ 正号 +3 3
- 负号 b=4;-b; -4
+ 加 3+3 6
- 减 5-3 2
* 乘 3*3 9
/ 除 3/3 1
% 取模 5%5 0
++ 自增（前） a=2;b=++a a=3;b=3
++ 自增（后） a=2;b=a++ a=3;b=2

-- 自减（后） a=2;b=a-- a=1;b=2
+ 字符串相加 "He"+"llo" "Hello"

  • 赋值运算符

    =、+=、-=、* = 、/=、%=

    int a,b,c; a = b = c = 3;

    int a = 3; a += 5;等同运算a = a + 5

  • 关系运算符

运算符 运算 范例 结果
== 相等于 4==3 false
!= 不等于 4!=3 true
< 小于 4<3 false
> 大于 4>3 true
<= 小于等于 4<=3 false
>= 大于等于 4>=3 false

  • 逻辑运算符

    逻辑运算符：&&(AND短路)、 ||(OR短路) 、 !(NOT非) 、 ^ (XOR异或) 、 &(AND与) 、 | (OR或)。

  • 位运算符

    位运算符有：左移( << )、右移( >> ) 、无符号右移( >>> ) 、位与( & ) 、位或( | )、位非( ~ )、位异或( ^ )

  • 移位运算符

    • 左移操作“<<”：将运算数的二进制码整体左移指定位数，左移之后的空使用“0”来补充

    • 右移操作“>>”：将运算数的二进制码整体右移指定位数，右移之后的空使用“符号位”来补充。正数用“0”补充，负数用“1”补充

    • 无符号右移">>>"：将运算数的二进制码整体右移指定位数，右移之后的空使用“0”来补充

  • 字符穿连接符

    • “+” 除用于算数加法运算外，还可以对字符串进行连接操作：

      • String s = "cd" +"itcast"; //cditcast

    • “+”运算符两侧的操作数中只要有一个String类型，一个是基本数据类型时，编译成class文件后，两个操作数就被连接成了一个新的String值

  • 三目运算符

    • “三目运算符”，语法格式：

      x ? y : z;

  • 操作符优先级

    ！ > 算术运算符 > 关系运算符 > && > || > 赋值运算符
